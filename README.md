# Chem-Dry of Richmond — Ops Command Center

This repo is the command center for the Chem-Dry of Richmond AI ops stack, built by Obsidian Labs for VACD INC (Brian Curran, CEO; Katie, marketing).

This repo is **not the code** — the code lives in [`chem-dry-stack`](https://github.com/labsobsidian/chem-dry-stack). This repo holds the project's memory: state, architecture, decisions, constants, and launch checklist.

## How to use this repo

Read `PROJECT_STATE.md` first. Everything else supports it.

- **`PROJECT_STATE.md`** — what's built, what's active, what's next
- **`ARCHITECTURE.md`** — how the pieces fit together and why
- **`CONSTANTS.md`** — IDs, URLs, endpoints (no secrets)
- **`DECISIONS.md`** — append-only log of architectural choices
- **`GO_LIVE_CHECKLIST.md`** — the pre-launch punch list
- **`.skills-version`** — which version of the Obsidian Labs skills library this project is built against
